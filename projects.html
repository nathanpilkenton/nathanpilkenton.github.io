    Projects

    Level

    Level is a service that helps teams track and improve their happiness and engagement, designed to be simpler and more transparent than exisitng options. (You can learn more about it here.) I built all of the front-end and back-end myself using Python/Django and HTML/CSS/JS.


    subscrippedoff

    I made subscrippedoff over a long weekend, because I was frustrated with how much money I was spending on subscription services and wanted to help other people see if they felt the same way. Django made it easier to manage all the products and prices, but the heavy lifting is in JQuery and HTML/CSS.


    Houdini

    Houdini was a Mac OS X application that allowed the user to easily manipulate "hidden files," a feature typically reserved for system use. I created it in high school, when I needed to delete some hidden files from a USB drive. It received over 350,000 downloads on MacUpdate, and was featured on Lifehacker in 2008. I stopped development in 2011.


    This website

    I made this website from scratch to learn HTML and CSS. It was first developed while I was in college and has been updated with new elements as my skills have grown. I also created all of the design elements myself (for better or worse).


    CFB Exchange

    The CFB Exchange was a web project I developed for fun in the summer of 2018 to learn Python and Django. It was a virtual stock exchange where users could buy or sell stock in college football teams. Prices were market-driven - the more users who buy a stock, the higher the price goes. At its peak it had over 100 users, but sadly, it's no longer online.
